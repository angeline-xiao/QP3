# QP3

This is the repository for the qualifying project on the paper: 
Charles Gaber, Kent Hanson, Sodam Kim, Jennifer Lund, Todd Lee, & Eleanor Murray. (Sept. 2024). “The Clone-Censor-Weight Method in Pharmacoepidemiologic Research: Foundations and Methodological Implementation”. en. In: Current Epidemiology Reports 11.3, pp. 164–174. issn: 2196-2995. doi: 10.1007/s40471-
024-00346-2.

ccw.R: This file contains all functions needed to run the testing. `ccw_sim()` is the main function for simulating the corticosteroid study as with measurements occuring in 30 day intervals from 0 days to 150 days. 

outputs.Rmd: This file created the models and outputs displayed in the results section of this report.
